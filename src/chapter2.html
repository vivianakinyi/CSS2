Chapter 2: Selectors

  • Selectors gives ability to apply set of styles to all elements of the same type.
  • They can also be use for narrow scope to apply style to an exact element.
  • Rules structure of selectors example h1: {color: red} where:
    • h1 is the selector. They are mostly elements but not always
    • {color: red} the declaration
  • There are two types of selectors: Class and Id selectors
    • Class selector uses a period eg .myClass{}.
    • Id selector uses the hash eg #myId{}.
    • The difference between the two is that:
      1. Id used only once i.e is unique
      2. Id cannot be combined like the class eg p.warning.help{}
      3. Id is used to enforce uniqueness
